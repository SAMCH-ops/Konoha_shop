# Projet de Site Web : KONOHA_SHOP

## Description

Nous avons décidé de créer un site web dédié à la vente d'articles pour otakus. Notre objectif est de fournir une plateforme où les fans de culture otaku peuvent acheter divers produits, notamment :

- **Mangas** : Une large sélection de mangas populaires et récents.
- **habits personnalisés** : Vêtements uniques inspirés de leurs séries préférées (T-shirt,Pull,Pantalon,etc...).
- **Articles de collection** : Figurines, posters et autres objets de collection.
- **Cosplays** : Costumes et accessoires pour les fans souhaitant incarner leurs personnages préférés.

## Fonctionnalités

- **Catalogue de Produits** : Une interface conviviale pour parcourir et rechercher des articles.
- **Système de Panier** : Les utilisateurs peuvent ajouter des articles à leur panier et passer à la caisse.
- **Gestion des Utilisateurs** : Inscription et connexion des utilisateurs pour gérer leurs commandes.
- **Base de Données MySQL** : Stockage des informations sur les produits et les utilisateurs.
- **Interface Réactive** : Conception responsive pour une expérience utilisateur optimale sur tous les appareils.

## Technologies Utilisées

- **Frontend** :
  - HTML
  - CSS
  - JavaScript

- **Backend** :
  - PHP
  - MySQL pour la gestion de la base de données

- **Outils de Développement** :
  - Git et GitHub pour le contrôle de version
  - Visual Studio Code comme éditeur de code
